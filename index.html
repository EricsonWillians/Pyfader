Pyfader

A simple pygame library that applies fade-in and fade-out effects to Surface objects.

Download this project as a .zip file Download this project as a tar.gz file

Fade easily!

With this open-source library you'll feel inspired to apply fade-in and fade-out effects to all your surfaces.

Use it by default!

The IFader class loads an image that can be drawn without fading effects. It's really simple!
